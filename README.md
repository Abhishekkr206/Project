# HTML and CSS Landing Page

A simple landing page created using HTML and CSS to practice layout, styling, and responsive design.

## Live Demo

[https://abhishekkr206.github.io/Project/](https://abhishekkr206.github.io/Project/)

## About

This project is a clean and modern landing page built using only HTML and CSS. It was created to practice core frontend concepts such as layouts, typography, spacing, and responsive design. The page includes sections like a hero banner, navigation bar, featured product area, and simple visual elements.

The goal of this project was to apply everything learned so far in HTML and CSS.

## Tech Stack

* HTML
* CSS

## Project Structure

```
project/
├── index.html       # Main landing page
├── style.css        # Styles
├── script.js        # Any small interactions
└── assets/          # Images and icons
```

## How to Use (Clone and Run)

1. Clone the repository

```
git clone https://github.com/Abhishekkr206/Project.git
```

2. Open the folder

```
cd Project
```

3. Run the project
   Open `index.html` in your browser.
